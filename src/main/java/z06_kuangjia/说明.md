# 框架

### SpringBoot
* 自动装配 
```

  @EnableAutoConfiguration和@ComponentScan两个注解。
  @ComponentScan注解的作用是标记 
  @EnableAutoConfiguration 下的 @Import({EnableAutoConfigurationImportSelector.class}) 便利所有jar包最终扫描的是所有参与项目的 classpath下META-INF/spring.factories的内容
  @EnableAutoConfiguration 下的@AutoConfigurationPackage 注解扫描 SpringBootApplication所在的Application类的包下的 所有 @Component类
  
  加载所有可加载的类 #会有一些类似@Condition的控制条件

```
* 启动原理
```
SpringApplication.run(AppDemo.class,args);
// ->
     public static ConfigurableApplicationContext run(Object[] sources, String[] args) {
        return (new SpringApplication(sources)).run(args);
    }
// 分为2部分
(1) 创建SpringApplication
SpringApplication sa= (new SpringApplication(sources)); //sources启动类
(2) 调用SpringApplication实例去运行run
sa.run(args);
```
