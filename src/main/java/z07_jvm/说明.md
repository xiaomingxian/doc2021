## 栈：
```
  方法本身:方法内部的代码
  方法内部的变量(基本变量和对象的引用)(基本变量eg:int,short...) #String,对象等都是在堆中，栈中存引用
 ```
 ## 方法区
 ```
 1.常量，静态变量，即时编译的代码
 
 2.运行时常量池(元数据[类的版本，字段，方法，接口相关信息的描述])
 
 //方法区是常量和静态变量 栈里是变量
 //方法区与永久带的去别：方法区是JVM中的概念,永久带是HotSport对方法区的实现
 //1.8以后用元空间(使用本地内存)取代永久带，之后就见不到方法区溢出
 
 ```
